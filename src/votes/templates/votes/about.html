For the curious about what I do, SEO stands for Search Engine Optimization, meaning I specialize in developing strategies to increase the rank of webpages when a user is searching for a certain keyword, such as "Jeff Chen." I figured I could use my knowledge to help all Jeff Chens rank at the same time in Google so I built a website featuring as many Jeff Chens as I could find.

For those who are curious about how I built the site, I built it using Django (Python) on AWS Elastic Beanstalk, S3, AWS Elasticsearch Services, PostgreSQL on AWS RDS, Route 53, Cloudflare, AWS SES, Bootstrap, Google Material Components, JQuery, SCSS, CSS, HTML. I also used other services like Google Search Console, Google Analytics, Google Optimize, Segment, Mixpanel, Hotjar, Mailgun, Sentry, Pingdom, AddThis, and more! Huge shoutout to Django's open-source community, GitHub repos, Youtube tutorials, and Stack Overflow for making this actually possible.
